# v0.2.0

## Fixes

- **Fix macOS `setRawMode` crash** — Claude Code (and other Node.js/Bun tools) failed with `setRawMode failed with errno: 1` inside the sandbox. The seatbelt profile was missing `(allow file-ioctl)`, which `tcsetattr()` needs to switch the terminal to raw mode.

## Improvements

- **GitHub repo URL in config header** — generated `.ai-jail` files now include a link to the project so users unfamiliar with the tool can find the documentation.
- **Binary distribution** — prebuilt binaries for Linux x86_64 and macOS ARM are now available via GitHub Releases, Homebrew, cargo install, and mise.

## Install

```bash
# Homebrew (macOS / Linux)
brew tap akitaonrails/tap && brew install ai-jail

# cargo install
cargo install ai-jail

# mise
mise use -g ubi:akitaonrails/ai-jail

# GitHub Releases (Linux)
curl -fsSL https://github.com/akitaonrails/ai-jail/releases/latest/download/ai-jail-linux-x86_64.tar.gz | tar xz
sudo mv ai-jail /usr/local/bin/

# GitHub Releases (macOS ARM)
curl -fsSL https://github.com/akitaonrails/ai-jail/releases/latest/download/ai-jail-macos-aarch64.tar.gz | tar xz
sudo mv ai-jail /usr/local/bin/
```
