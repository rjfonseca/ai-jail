# v0.4.2

## Fixes

- **Fix Landlock blocking bwrap mount namespace setup** — bwrap calls `mount --make-rslave /` during startup, which requires Landlock to allow access to `/` for path resolution. The Landlock rules only included subdirectories (`/usr`, `/bin`, etc.) but not `/` itself, causing `Failed to make / slave: Operation not permitted`. Added `/` as read-only to the Landlock ruleset in both normal and lockdown modes.
